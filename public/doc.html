Rotas da API

Esta documentação fornece uma visão geral das rotas disponíveis na API para a gestão de livros.

  • GET /books - Retorna toda a lista dos livros registados.
  • GET /books/{id} - Retorna as informações de um livro específico, a partir do seu ID.
  • POST /books - Adiciona um novo livro no sistema.
  • PUT /books/{id} - Atualiza as informações de um livro existente, a partir do seu ID.
  • DELETE /books/{id} - Remove um livro do sistema, a partir do seu ID.

Nota:

Para registar um novo livro, os dados necessários incluem: título, autor, editora e edição. Estes campos são obrigatórios para o registo de um novo livro.
